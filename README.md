## ✨ 關於我們

『馡夢』原意為法文手工製作，我們是一群熱愛美食的大學生，希望可以將手作溫度傳達給每位支持我們的人❣️

堅持使用天然食材，不添加防腐劑，為您帶來最純粹的美味體驗。每一份點心都由我們親手製作，嚴格控制品質，確保每一口都能品嚐到最純正的美味。
